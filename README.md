# GUI 測試


## 安裝

```bash
pip install websockets
```

## `websocket-demo`

先在 Terminal 執行 `server.py`:

```bash
cd websocket-demo
python3 server.py
```

接著直接在瀏覽器 (chrome) 打開 `websocket-demo/` 中的 `index.html`，就可以開始囉～

在跑 IAT 的時候，Terminal 裡面會印出 browser 和 Python (server) 溝通的內容。
